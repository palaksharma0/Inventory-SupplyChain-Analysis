# 📦 Inventory & Supply Chain Analysis

## 📌 Project Overview
This project performs exploratory data analysis (EDA) on an Inventory and Supply Chain dataset to uncover insights related to stock levels, supplier performance, order fulfillment, and demand patterns.

## 🗂️ Dataset
- **File:** `Inventory_SupplyChain_Dataset.csv`
- **Domain:** Supply Chain Management
- **Key Features:** Product inventory levels, supplier details, order quantities, lead times, and demand data

## 🎯 Objectives
- Analyze inventory trends and stock movement
- Identify supply chain bottlenecks
- Evaluate supplier performance
- Understand demand patterns across categories

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- Data Visualization (Matplotlib, Seaborn)
- Jupyter Notebook
- Excel / CSV
